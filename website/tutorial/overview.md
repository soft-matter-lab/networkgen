# Tutorial Overview

This tutorial section is the guided path through NetworkGen.

Use it when you want to build a network step by step and understand which settings matter first, which settings can be left at their defaults, and when it is worth moving from a basic setup to an advanced one.

## How this tutorial is organized

The tutorial is split into four parts:

1. [Supported Networks](./supported-networks) gives the high-level map of the network families covered here.
2. [Lattice Networks - Basics](./lattice-networks/basics) walks through the simplest hexagonal-lattice setup.
3. [Random Networks - Basics](./random-networks/basics) starts from the default random generator, then moves into manual strand and perbond control.
4. [Adding Defects and Disorder](./defects-disorder/overview) shows how to add lattice disorder or explicit void defects after the base network is working.

## Suggested reading order

If you are new to NetworkGen, read the pages in this order:

1. Supported Networks
2. Lattice Networks - Basics
3. Random Networks - Basics
4. Random Networks - Advanced
5. Defects & Disorder

## Before you start

These tutorial pages assume that:

- NetworkGen is installed and on your MATLAB path
- you are comfortable editing a short MATLAB script
- you know whether you want a lattice network or a random network before fine-tuning the strand statistics

For installation details, see [Installation](/docs/install). For the full API, see [Full Settings Reference](/docs/full-reference).

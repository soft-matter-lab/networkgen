# For Developers

NetworkGen is built around four MATLAB classes. Instantiate `network`, configure its properties, then call `generateNetwork()`. The other three classes are automatically instantiated as subclass properties.

Click any class to highlight its connections. Reference fields shown in green or purple link to subclasses.
